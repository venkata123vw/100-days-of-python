# Day 9 — Dictionaries & Blind Auction

## What I Learned
- Creating and updating dictionaries `{key: value}`
- Accessing dictionary values using keys
- Iterating over dictionaries with for loops
- Passing dictionaries into functions as arguments
- Using while loops with boolean flags to control flow
- Clearing screen output with `print("\n" * 100)`
- Writing logic independently before seeing the solution

## Project Built
**Blind Auction Program** — A command-line auction where multiple
bidders secretly enter their name and bid amount. The program
stores all bids in a dictionary, clears the screen between
bidders, and finds the winner by comparing all bids.
